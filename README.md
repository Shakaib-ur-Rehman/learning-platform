# Learning Platform

This repository is created for learning and practicing full-stack development.

## Purpose
The goal of this project is to build a learning platform where users can:
- Browse courses
- Enroll in learning content
- Track their progress

This project is mainly for educational purposes and backend/frontend practice.

## Tech Stack (Planned)
- Frontend: React / Next.js
- Backend: Node.js (NestJS)
- Database: PostgreSQL
